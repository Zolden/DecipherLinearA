# Линейное письмо А: воспроизводимая статистика (по-русски)

Английская витрина — в [README.md](README.md). Главные документы:

- [linear_a_full_report_v2.md](linear_a_full_report_v2.md) — полный отчёт
  (§0–§BF): методика, все тесты, честные негативы.
- [article_popular_ru.md](article_popular_ru.md) — научно-популярный
  пересказ для неспециалистов (англ. версия:
  [article_popular_en.md](article_popular_en.md)).
- [FAQ.md](FAQ.md) — вопросы и ответы (EN).

Кратко: мы не «переводим» линейное А. Мы замораживаем корпус, строим
нуль-модели и отбираем закономерности, которые не объясняются случайностью:
переносимость чтений из линейного B, слой имён и топонимов, доживших до
греческих архивов, калиброванная морфология (-JA, A-, PI-), «слот
географии» в формуле возлияний, товарная метрология. Всё воспроизводится
скриптами при seed=42 и PYTHONHASHSEED=0.
